## 🧠 Overview

**SafeRoute AI** is a safety-first navigation platform that leverages **AI, real-time data, and community intelligence** to recommend the safest routes instead of the fastest ones.

It addresses a critical limitation in existing navigation systems by introducing **risk-aware routing**.

---

## ⚠️ Problem

Traditional navigation systems:
- Optimize only for **time and distance**
- Ignore **crime-prone areas and unsafe environments**
- Provide **no real-time safety insights**

This results in users being unknowingly exposed to **avoidable risks**, especially during night travel.

---

## 💡 Solution

SafeRoute AI integrates a **multi-factor safety scoring engine** that evaluates routes using:

- Crime density data  
- Infrastructure conditions (lighting, roads)  
- Emergency service proximity  
- Real-time community reports  

Users receive **ranked routes with safety insights**, enabling informed decisions.

### ⚙️ Tech Stack

#### 🧠 AI & Intelligence
- **Groq AI (LLM)** — real-time safety insights and explanations  
- **Natural Language Processing (NLP)** — contextual risk interpretation  
- **Predictive Analytics** — time-based safety forecasting  

---

#### 🗺️ Maps & Location Services
- **Google Maps JavaScript API** — interactive maps and visualization  
- **Google Directions API** — route generation and navigation data  
- **Google Places API** — nearby emergency services and locations  
- **Heatmap Layer** — visualization of high-risk zones  

---

#### ⚙️ Backend
- **Python 3.x** — core application logic  
- **Flask** — lightweight web framework for API development  
- **Flask-SocketIO** — real-time communication and alerts  
- **REST APIs** — structured communication between frontend and backend  

---

#### 🎨 Frontend
- **HTML5** — application structure  
- **CSS3** — responsive and modern UI design  
- **JavaScript (Vanilla)** — core client-side logic  
- **Web Speech API** — voice-based interaction and narration  

---

#### 💾 Data & Storage
- **SQLite** — lightweight, embedded database  
- Stores:
  - User data  
  - Community reports  
  - Incident and safety records  

---

#### 🔌 Real-Time Communication
- **Socket.IO** — real-time updates, SOS alerts, and live dashboard  

---

#### ☁️ Deployment & Infrastructure
- **Render** — backend and frontend hosting

- **Cloud-based architecture** — scalable and cost-efficient
